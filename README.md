# 即梦浮动输入框

这是一个本地解压加载的 Chrome MV3 扩展，当前支持这些即梦页面：

- `https://jimeng.jianying.com/ai-tool/generate*`
- `https://jimeng.jianying.com/ai-tool/canvas*`

## 功能

- 在页面右下角注入一个浮动输入框
- 支持前缀提示词 + 用户输入 + 后缀提示词的拼接发送
- 支持前缀、后缀、动态提示词各自单独开关，默认关闭
- 关闭时只隐藏对应区块，并且不参与拼接或替换
- 支持动态提示词，格式必须为 `"key":["val"]` 或 `"key":["val1","val2"]`
- 支持动态提示词多值循环发送，例如 `"风格":["财阀","时尚","校园"]`
- 兼容这些输入形态：逐行键值对、单行逗号拼接多个键值对、以及外层 `{...}` 包裹
- 支持浅色/深色模式显示
- 输入框内容和开关状态会保存到 IndexedDB，刷新页面后自动恢复
- 多值参数按位置配对展开，并按固定间隔自动逐条发送
- 循环发送过程中，发送按钮会切换为“终止”，可随时停止后续批次
- 支持通过输入框和应用按钮设置循环发送间隔时长
- 保留官方输入框，不改原页面布局
- 发送时把浮层文本写入官方输入框，再点击官方发送按钮
- 支持 `Enter` 发送，`Shift+Enter` 换行
- 支持折叠浮层
- 支持拖动展开态浮层头部，调整浮层位置
- 支持拖动折叠态小浮窗，调整浮层位置
- 浮层位置会保存到 IndexedDB，刷新页面后自动恢复

## 本地加载

1. 打开 Chrome，访问 `chrome://extensions`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `/Users/zhengjiyuan/Desktop/jimeng`
5. 打开即梦生成页或画布页验证效果

## 已知限制

- 仅支持当前已适配的即梦生成页与画布页
- 不做双向同步
- 不改官方接口，完全复用页面已有发送流程
- 即梦前端类名如果后续大改，可能需要更新适配逻辑
